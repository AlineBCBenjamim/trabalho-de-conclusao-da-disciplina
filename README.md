# PGATS 3 - Programação para automação de testes.

## Introdução
Projeto desenvolvido em `JavaScript` e `Mocha` para práticar a lógica de programação e as execuções de teste, foi utilizado:

- Criação de classe;
- Construtor;
- Criação de Métodos;
- Estrutura Condicional (if/ else);
- Objetos;
- Teste Automatizados.

## Descrição do Projeto
Serviços de Pagamento e Consulta. O sistema consiste em uma classe responsável por:

- Realizar pagamentos;
- Armazenar pagamentos em uma lista;
- Consultar o último pagamento realizado;
- Categorizar automaticamente os pagamentos.

A categorização ocorre da seguinte forma:

- Valor maior que R$ 100,00 → categoria Cara.
- Valor menor que R$ 100,00 → categoria Padrão.

## Rodar o projeto
- Rode o console na pasta em que você baixou os arquivos;
- Insira o comando npm install;
- Depois, o comando npx mocha para rodar.
